# 概述

ElasticAmbari为[Ambari](https://ambari.apache.org/)的一组自定义服务，为[Elastic](https://www.elastic.co/)的产品提供安装和管理的支持。

# 特性

- 支持[Elastic](https://www.elastic.co/)全产品系列
- 支持[Elastic](https://www.elastic.co/)各产品的各个版本，不要求特定版本
